1. more は何をするのか
　一度に１画面ずつファイルの内容を表示する。

2. more はどんな仕組みなのか

+--->入力から２４行を出力する
|+-->[more?]  メッセージを表示する
||   [Enter], space, "q" のどれかが入力される
|+-- [Enter]なら、１行先に進める
+----space なら
     "q" なら--> 終了する

=================
プログラミング問題 ( more03 実装 )
画面の最下部にナンパーセントまで表示したか反転表示する。
即時入 ( Enterを押さないで応答する。 )
ユーザがキーを押した後、more? promptを消す。
