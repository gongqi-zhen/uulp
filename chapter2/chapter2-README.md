ユーザ、ファイル、マニュアルー最初はwhoから

UnixからUnixのことを学ぶには
- マニュアルを読む
- マニュアルを検索する ( $ man -k )
- .hファイルを読む
- 関連項目リンクをたどる ( SEE ALSO on man )

1. who は何をするのか
	システムのログファイルを読んで、現在のユーザのリストを出力する。

2. who はどんな仕組みなのか

	utmpをオープンする
+-> レコードを読む     --+
+-- レコードを表示する <-+
	utmpをクローズする

UnixからUnixのことを学ぶには
このテクニックによって、コマンドが何をするのかがわかり、ヘッダファイルからは
コマンドがデータ構造がわかった。
しかし、何かを本当に理解できているかどうかは、自分で試してみなければわからない
